## 🎯 使用技巧

### 技巧1：信息越详细，分析越准确

提供的信息越完整，Agent的分析和建议就越精准。

**最少需要的信息**：
- 出生时间（用于命理分析）
- 当前职位和行业（用于职场建议）

**推荐提供的信息**：
- 完整的个人信息、命理信息、职场信息

### 技巧2：可以随时提问

无需每次都提供完整信息。Agent会记住你的信息，后续对话可以直接提问。

### 技巧3：善用具体场景描述

与其问"如何提高工作效率？"
不如问"我是产品经理，每天要开很多会，如何提高工作效率？"

### 技巧4：可以追问任何细节

如果对某个建议不满意或想了解更多，可以继续追问。

**示例**：
```
你：如何与ENFP类型的同事沟通？
AI：[给出建议]
你：如果他是我的上司呢？
你：如果我们在项目中有分歧呢？
你：能给我一些具体的话术吗？
```

## ❓ 常见问题

### Q1：提供的信息会被保存吗？

**A**：你的信息仅用于当前的对话会话，不会长期保存或共享。Coze平台有严格的数据保护政策。

### Q2：命理分析准确吗？

**A**：命理分析基于传统文化视角，**仅供娱乐参考**。实际决策请结合现实情况。

### Q3：MBTI不知道怎么办？

**A**：MBTI类型是可选的，可以跳过。如果想知道自己的MBTI，可以：
- 搜索"MBTI测试"在线测试
- 告诉Agent"帮我分析一下我的MBTI类型"

### Q4：可以用于商业决策吗？

**A**：Agent的建议可以参考，但不应作为唯一决策依据。重要商业决策请结合专业建议和实际情况。

### Q5：每天都可以用吗？

**A**：可以！你可以随时提问，获取实时建议和运势分析。
