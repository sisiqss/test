# Coze平台使用指南 - 职场情绪充电站

## 🤔 为什么访问不到Web主页？

### 重要说明

**Coze平台不是一个传统的Web应用部署平台！**

Coze是一个**AI对话机器人平台**，它有自己固定的界面风格和交互方式：

- ❌ **不支持**自定义完整的Web前端页面（HTML/CSS/JS）
- ❌ **不支持**自定义主页布局
- ✅ **支持**在聊天界面中与Agent对话
- ✅ **支持**通过系统提示词定制Agent的行为

### 你访问的URL

当你访问 `http://wdssb8q7gh.coze.site/` 时，你看到的是：

1. **Coze平台的标准聊天界面**
2. **聊天输入框**
3. **Agent的回复区域**

而不是我之前创建的静态HTML页面（那些页面是为本地HTTP服务器设计的）。

## ✅ 正确的使用方式

### 方式1：通过Coze聊天界面使用（推荐）

访问 `http://wdssb8q7gh.coze.site/` 后：

#### 第一步：发送任意消息

在聊天框中输入任意内容，例如：
- "你好"
- "开始"
- 或直接输入你的需求

#### 第二步：查看欢迎语

Agent会自动显示欢迎语：

```
🌟 欢迎来到职场情绪充电站！🌟

我是你的专属职场导师，将为你提供命理分析、职场建议、MBTI性格分析和情绪支持服务。

为了给你提供最贴心的个性化服务，请告诉我以下信息：

📋 【基本信息】
1. 你的姓名（可以只告诉我昵称）
2. 你的性别（男/女）
3. MBTI类型（可选，如INTJ、ENFP）

🎯 【命理信息】（用于运势分析）
4. 出生年月日时（如：1990年03月15日08时）
5. 出生地理位置（如：北京，可留空使用当前位置）

💼 【职场信息】（用于职业建议）
6. 当前地理位置（用于天气和穿搭建议）
7. 行业（如：互联网、金融、教育）
8. 职位（如：产品经理、软件工程师）
9. 职级（如：P6、P7、高级、经理）

你可以一次性告诉我，也可以逐条提供～ 😊
```

#### 第三步：提供信息

你可以一次性提供所有信息，也可以分步提供。Agent会自动识别并收集。

**示例（一次性提供）**：
```
我叫小明，男，INTJ。
1990年3月15日8点出生在北京。
现在在上海，互联网行业，产品经理，P6职级。
```

#### 第四步：获取完整报告

Agent会自动生成4个板块的报告：
- 📚 我的人生解读（八字分析）
- ✨ 每日运势（今日宜忌）
- 💼 职场大势（职业建议）
- 👔 每日穿搭（穿搭建议）

#### 第五步：日常使用

之后你可以随时询问任何问题：

**命理分析**：
- "请帮我分析今天的运势"
- "我的五行怎么样？"

**MBTI分析**：
- "我是INTJ，在职场中应该注意什么？"
- "分析一下我的MBTI性格"

**运势趋势图**：
- "生成我本月的运势趋势图"
- "我想看下半年的运势走势"

**人际关系**：
- "如何与ENFP类型的同事更好沟通？"
- "和领导关系紧张怎么办？"

**职业转型**：
- "我想从技术岗转产品经理，有什么建议？"
- "如何判断是否适合职业转型？"

### 方式2：直接使用快捷指令

你也可以直接输入你的需求，Agent会根据系统提示词自动处理：

**示例**：
```
请帮我分析今天的运势
```
或
```
生成我本月的运势趋势图
```

## 🎨 关于"功能按钮"

### Coze平台的限制

Coze平台目前**不支持**在聊天界面中添加自定义的功能按钮。

但是，你可以通过以下方式实现类似效果：

### 方案1：使用Coze的"推荐问题"功能（如果支持）

在Coze平台的Agent配置中，可能有"推荐问题"或"快捷回复"的设置项。

**推荐的快捷问题**：
1. 🔮 请帮我分析今天的运势
2. 🧠 我是INTJ，在职场中应该注意什么？
3. 📈 生成我本月的运势趋势图
4. 👥 如何与不同性格的同事沟通？
5. 💼 我想进行职业转型，有什么建议？

### 方案2：在欢迎语中提示

Agent已经在欢迎语中清楚地列出了所有可用功能，用户可以参考提示进行提问。

### 方案3：创建独立的Web界面（高级）

如果你必须要有Web界面和功能按钮，需要：

#### 选项A：部署到自己的服务器

不使用Coze平台，将Agent部署到自己的服务器上，然后使用我创建的静态HTML页面。

详细步骤见 [docs/SHARING_GUIDE.md](SHARING_GUIDE.md)

#### 选项B：使用Coze API + 自建前端

使用Coze提供的API接口，自己开发前端界面。

**优点**：
- 完全自定义UI
- 可以添加任意功能按钮

**缺点**：
- 需要额外的开发工作
- 需要自己的服务器

## 💡 常见问题

### Q1：为什么我看不到功能按钮？

**A**：Coze平台是聊天机器人平台，不支持自定义功能按钮。你需要通过文字输入来使用各项功能。

### Q2：我创建的HTML页面在哪里？

**A**：那些HTML页面位于 `src/static/` 目录，是为**本地HTTP服务器**设计的，在Coze平台上不会被使用。

### Q3：如何在Coze上使用这个Agent？

**A**：
1. 访问 `http://wdssb8q7gh.coze.site/`
2. 在聊天框中发送任意消息
3. 按照欢迎语提示提供信息
4. 获取个性化报告
5. 继续对话，享受各项功能

### Q4：我想要有功能按钮，怎么办？

**A**：你有两个选择：
1. 适应Coze平台的聊天界面（推荐，最简单）
2. 部署到自己的服务器，使用自建的Web界面（需要额外工作）

### Q5：分享给其他人方便吗？

**A**：在Coze平台上非常方便，直接分享 `http://wdssb8q7gh.coze.site/` 这个链接即可。

## 🎯 总结

**在Coze平台上的最佳实践**：

1. ✅ 接受Coze平台的聊天界面风格
2. ✅ 利用系统提示词实现欢迎语和功能引导
3. ✅ 通过文字指令使用各项功能
